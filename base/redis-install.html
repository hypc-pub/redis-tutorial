Redis安装

Ubuntu下安装

执行以下命令安装redis服务：

$ sudo apt-get install -y redis-server
$ sudo systemctl start redis-server.service

使用Docker运行redis

$ docker run -dit -p 6379:6379 --name redis redis:latest
$ docker logs -f redis
